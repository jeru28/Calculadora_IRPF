# Calculadora IRPF / IRPFM 2026

Calculadora em HTML único para simular cenários de IRPF e IRPFM em 2026, com foco em:

- pró-labore
- dividendos por empresa
- aluguéis
- rendimentos tributáveis e isentos
- deduções legais
- comparação entre tributação tradicional e IRPFM

## Arquivo principal

- `calculadora_irpf_gov_style.html`

## Como usar

1. Abra o arquivo `calculadora_irpf_gov_style.html` no navegador.
2. Preencha os dados das empresas, rendimentos e deduções.
3. Acesse a aba de resultados para ver a comparação entre as trilhas de cálculo.

## Stack

- HTML
- CSS embutido
- React 18 via CDN

## Observação

Esta calculadora é uma estimativa de planejamento tributário e não substitui orientação contábil ou jurídica especializada.
